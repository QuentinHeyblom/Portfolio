# Portfolio

Bienvenue sur le dépôt de mon nouveau portfolio professionnel.

## À propos
Ce projet a pour but de recréer entièrement mon ancien site (réalisé sous WordPress) vers une architecture et un look plus moderne, en utilisant **React** et **Vite.js**.

## Technologies utilisées
Ce projet utilise les technologies web suivantes pour garantir performance et maintenabilité :
- **React 19** : Bibliothèque JavaScript pour l'interface utilisateur.
- **Vite** : Outil de construction rapide et serveur de développement.
- **React Router DOM 7** : Gestion du routage coté client.
- **ESLint** : Analyse statique du code pour assurer la qualité.

## Auteur

## Installation et démarrage

```bash
# Installation des dépendances
npm install

# Lancer le serveur de développement
npm run dev
```
